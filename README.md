# Project
School project to emulate Airbnb

To emulate Airbnb as closely as possible, we need to collect tourism-related data. We must use open datas from https://www.data.gouv.fr/ <br>
To meet this requirement, we are using DATAtourisme, an API that aggregates tourims-related data.<br>
Here are the API documentation links:<br>
https://api.datatourisme.fr/v1/docs <br>
https://api.datatourisme.fr/v1/swagger/
